POS Pizza Modifiers

POS Pizza Modifiers module is a useful odoo addons for customizing your POS Restaurant sales for Pizza and other foods.

Key Features

  • User can create Modifier Product and modifier groups.
  • User allow to create and Add Side Product along with main product.
  • When display pieces checkbox enable from backend then user can select the Portion of pizza in pos session.
